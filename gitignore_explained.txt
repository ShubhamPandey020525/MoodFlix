MoodFlix - Gitignore Reference (Personal Notes)

I have updated the .gitignore to keep the repo clean and professional. Here is a quick breakdown for future reference:

1. Sensitive Info (.env, .env.*)
   - TMDB API keys aur secrets isi mein hain.
   - Most important point: Inhe kabhi public mat karna, isliye ye ignored hain.

2. Data & Datasets (*.csv)
   - Large movie datasets (like movies_cleaned.csv) ignored hain.
   - GitHub par badi files se bachne ke liye ye zaroori hai.

3. AI Models (*.pkl, *.h5)
   - Trained AI models binary format mein hote hain aur kaafi space lete hain.
   - Ye files bhi ignored hain for better repo management.

4. Dependencies (node_modules)
   - Standard practice: Libraries ko 'npm install' se wapas laya ja sakta hai, so no need to push.

5. Build Output (dist, dist-ssr)
   - Production-ready files hain jo build command se generate hoti hain.

6. Legacy Configs (.lovable/)
   - Purane tool ki configurations hain, naye setup mein inki zaroorat nahi hai.

7. System/Editor Files (.vscode, .DS_Store)
   - Editor settings aur OS temporary files ignored hain to keep the workspace professional.

Summary: Everything is set up to protect my API keys and keep the repository lightweight.
